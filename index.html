Tentang Saya

Indramayu, 11 Oktober 2002

Saya adalah lulusan D4 Rekayasa Perangkat Lunak dengan minat besar pada System Analysis, Quality Assurance, dan Technical Writing. Saya memiliki pengalaman dalam menganalisis kebutuhan pengguna, membuat dokumentasi teknis dan fungsional, serta melakukan pengujian aplikasi menggunakan metode manual maupun tools seperti JMeter dan OWASP ZAP. Saya teliti dalam mengidentifikasi masalah, mampu menyusun dokumentasi yang terstruktur, dan senang berkolaborasi dalam tim untuk memastikan keandalan serta kepuasan pengguna terhadap perangkat lunak.

Pada tahun terakhir perkuliahan, saya menyelesaikan penelitian mengenai Analisis Sentimen menggunakan model machine learning dan deep learning berbasis Transformers. Pengalaman ini membantu saya mengasah kemampuan berpikir analitis, menulis laporan teknis, serta memahami integrasi antara teknologi dan sistem informasi. saya.

Saat ini saya mencari kesempatan untuk memulai karier sebagai System Analyst, QA, atau Technical Writer, dengan tujuan untuk mendukung proses pengembangan perangkat lunak melalui analisis sistem yang tepat, dokumentasi yang jelas, dan pengujian yang komprehensif. Saya berkomitmen untuk terus belajar teknologi baru dan meningkatkan keterampilan profesional saya.
