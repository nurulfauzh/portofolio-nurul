Tentang Saya

Indramayu, 11 Oktober 2002

Saya adalah lulusan D4 Rekayasa Perangkat Lunak dengan minat besar pada System Analysis dan Quality Assurance. Saya memiliki pengalaman dalam menganalisis kebutuhan pengguna, membuat dokumentasi, serta melakukan pengujian aplikasi menggunakan metode manual maupun dengan tools seperti JMeter dan OWASP ZAP. Saya teliti dalam mengidentifikasi masalah, serta senang berkolaborasi dalam tim untuk memastikan keandalan dan kepuasan pengguna terhadap perangkat lunak.

Pada tahun terakhir perkuliahan, saya menyelesaikan penelitian tentang Analisis Sentimen dengan menggunakan berbagai model machine learning dan deep learning, termasuk arsitektur berbasis BERT. Pengalaman ini semakin mengasah kemampuan berpikir analitis dan pemecahan masalah saya.

Saat ini saya mencari kesempatan untuk memulai karier sebagai System Analyst atau QA Engineer, sekaligus terus mempelajari teknologi baru dan meningkatkan keterampilan profesional saya.
